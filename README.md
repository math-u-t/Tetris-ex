# テトリス Chrome拡張機能

Google Chromeで遊べるテトリスゲームの拡張機能です。

## ファイル構成

```
Tetris-ex/
├── manifest.json      # 拡張機能の設定ファイル
├── popup.html         # ゲーム画面のHTML
├── styles.css         # スタイルシート
├── tetris.js          # ゲームロジック
├── icon16.png         # 16x16 アイコン
├── icon48.png         # 48x48 アイコン
└── icon128.png        # 128x128 アイコン
```

## インストール方法

1. このリポジトリをクローンまたはダウンロードします
2. Google Chromeを開きます
3. アドレスバーに `chrome://extensions/` と入力してEnterキーを押します
4. 右上の「デベロッパーモード」をONにします
5. 「パッケージ化されていない拡張機能を読み込む」ボタンをクリックします
6. このフォルダ（Tetris-ex）を選択します
7. 拡張機能がインストールされます

## 遊び方

1. ブラウザのツールバーにあるテトリスアイコンをクリックします
2. 「ゲーム開始」ボタンをクリックしてゲームを始めます

## 操作方法

- **←** : ミノを左に移動
- **→** : ミノを右に移動
- **↑** : ミノを右回転
- **Z** : ミノを左回転
- **↓** : 落下速度を加速（押している間）
- **スペース** : ミノを一番下まで落とす（ハードドロップ）

## ゲームルール

- 落ちてくるミノ（テトリミノ）を操作してラインを揃えます
- 横一列が揃うとそのラインが消えてスコアが加算されます
- 10ライン消すごとにレベルが上がり、落下速度が速くなります
- ミノが一番上まで積み上がるとゲームオーバーです

## 技術仕様

- **対応ブラウザ**: Google Chrome
- **使用技術**: HTML5 Canvas, JavaScript, CSS3
